# mailrocket

Email newsletter backend

## Getting Started

+ Add and install project dependencies

  ```
  cargo add actix-web
  cargo add tokio --features "macros rt-multi-thread"
  ```

+ Compile project

  ```
  cargo check
  ```

+ Run web server

  ```
  cargo run
  ```

+ Verify application (in a separate tab)

  ```
  curl 127.0.0.1:8080
  ```
